Esperiahotelbook

The EsperiaHotelBook is an app for Android Device. With it the user can book for a room at Esperia Hotel in Kavala of Greece.

View on GitHub Download .zip Download .tar.gz

Περίληψη

Πρόκειται για μια εφαρμογή για Android συσκευές, όπου ο χρήστης μπορεί να κάνει κράτηση σε ένα συγκεκριμένο ξενοδοχείο με ένα απλό SMS. Πρόκειται για μια δοκιμαστική έκδοση. Τυχαία έχει επιλεχθεί ένα πραγματικό ξενοδοχείο της Καβάλας, το Esperia Hotel.

Σκοπός

Η εφαρμογή όντως πραγματοποιεί το σκοπό της, αλλά το SMS δεν στέλνεται σε αληθινό αριθμό του εν λόγω ξενοδοχείο (δεν υπάρχει καμία επικοινωνία με τη βάση δεδομένων του ξενοδοχείου), αλλά σε ένα δοκιμαστικό αριθμό. Η χρέωση είναι αληθινή.

Εν κατακλείδι

Είναι μια εφαρμογή για γρήγορη κράτηση και απευθύνεται στον χρήστη που δεν διαθέτει σύνδεση internet στη συσκευή του, ώστε να πραγματοποιήσει online κράτηση.
